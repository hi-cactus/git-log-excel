### Installing

```bash
$ npm install git-log-excel  -g
```

### Usage

```bash

cd your-git-project

glog
```

filter commit email

```bash
cd your-git-project

glog --email=test@test.email
```

export excel path

```bash
cd your-git-project

glog --email=test@test.email --exportPath=../
```
